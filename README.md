# Sistema de Gestão e Controle de Estoque — Front‑end

Este é o módulo **front-end** do Sistema de Gestão e Controle de Estoque, responsável pela interface de usuário para cadastro de produtos, pedidos, controle de estoque e relatórios.

---

## 📋 Tecnologias usadas
- HTML  
- CSS  
- JavaScript  

---

## 🖥️ Instalação e execução local

1. Clone o repositório ou baixe os arquivos:
   ```bash
   git clone https://github.com/Auriacarvalho/Sistema-de-Gest-o-e-Controle-de-Estoque.git
   ```

2. Navegue até a pasta do front-end:
   ```bash
   cd Sistema-de-Gest-o-e-Controle-de-Estoque/Projeto\ PI\ Front-end/projeto_frontend
   ```

3. Abra o arquivo `login.html` em seu navegador (Chrome, Firefox, Edge, etc.).  
   Não há backend ou servidor necessário — tudo roda localmente.

---

## 🔐 Acesso
- **Usuário:** `admin`  
- **Senha:** `1234`  

Após o login, você será redirecionado para a página principal (`index.html`).

---

## ✅ Funcionalidades disponíveis
- Cadastro de produtos  
- Visualização de pedidos  
- Gerenciamento de estoque  
- Controle financeiro  
- Geração de relatórios  

---

## 🔄 Navegação
- O botão **"Voltar"** retorna para a página principal (`index.html`).  
- O botão **"Sair"** encerra a sessão e volta para `login.html`.  

---

## ⚠️ Observações
- O sistema front-end é **estático** e **não possui integração com banco de dados**.  
- Para adicionar persistência ou lógica de backend, é necessário implementar um servidor (Node.js, PHP, Python, etc.) e ajustar o front-end para consumir APIs.
